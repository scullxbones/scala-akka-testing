## Scala / Akka testing survey
==================

Tips and tricks in Scala testing; Lessons learned in Akka testing

Covers:
* Scalatest/Specs2/ScalaCheck
* Deeper dive on Scalatest
* Mocking in Scala
* Akka TestKit
* Unit testing actors
* Integration testing actors
* Multi-JVM testing a cluster
